# Чеклісти для тестування

## Модуль: Управління парами

### Функціональність

- [ ] Можна створити нову пару
- [ ] Пара зберігається у системі
- [ ] Пара відображається у розкладі
- [ ] До пари можна додати викладача
- [ ] До пари можна додати групу
- [ ] Можна редагувати пару
- [ ] Можна видалити пару якщо вона ще не в розкладі

### Негативні сценарії

- [ ] Не можна створити пару без назви
- [ ] Не можна створити пару без групи
- [ ] Не можна створити пару без викладача
- [ ] Не можна ввести текст у поле годин
- [ ] Не можна ввести відʼємну кількість годин

### UI/UX

- [ ] Форма створення відкривається
- [ ] Кнопка створення працює
- [ ] Помилки відображаються коректно
- [ ] Повідомлення про успіх відображається
