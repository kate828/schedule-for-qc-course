# Чеклісти для тестування

## Модуль: Управління парами

### Функціональність

- [ ] Можна створити пару з валідними даними
- [ ] Після створення пара з’являється у списку пар
- [ ] Можна редагувати існуючу пару
- [ ] Зміни після редагування зберігаються
- [ ] Можна змінити викладача,назву,тип, к-сть годин при редагуванні
- [ ] Можна видалити існуючу пару
- [ ] Після видалення пара зникає зі списку

### Негативні сценарії

- [ ] Не можна створити пару без назви
- [ ] Не можна створити пару без групи
- [ ] Не можна створити пару без викладача
- [ ] Не можна ввести текст та відʼємну кількість у поле годин

### UI/UX

- [ ] Форма створення відкривається
- [ ] Кнопка створення працює
- [ ] Помилки відображаються коректно
- [ ] Повідомлення про успіх відображається
